About Regnum Mathematica

Regnum Mathematica is a royal mathematical computation system built with pure HTML, CSS, and JavaScript — no frameworks, no backend.

It features seven classical mathematical algorithms: Fibonacci, Lucas, Tribonacci, Collatz Conjecture, Bernoulli Numbers, Euclidean Division, and GCD & LCM computation.
